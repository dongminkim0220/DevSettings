## Init Github repository & Connect to Local

[github]

[local]
  
  2. git init
  
  3. git status
  
  4. git add .
  
  5. git commit -am "message"
  
  6. git remote add origin "\<url of github\>"
  
  7. git remote -v
  
  8. git push origin master
